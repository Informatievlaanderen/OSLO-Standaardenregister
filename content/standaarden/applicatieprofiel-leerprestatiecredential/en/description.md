This vocabulary collects the terms defined around learning credentials. The application profiles around <a href="https://data.vlaanderen.be/doc/applicatieprofiel/leerinschrijfcredential/">Learning Enrolment Credentials</a> and <a href="https://data.vlaanderen.be/doc/applicatieprofiel/leerprestatiecredential/">Learning Performance Credentials</a> are derivatives of this vocabulary. This vocabulary is based on the <a href="https://github.com/european-commission-empl/European-Learning-Model">European Learning Model</a>.
  The latter in turn uses the <a href="https://www.w3.org/TR/vc-data-model/">W3C Verifiable Credentials Data Model</a> as an overarching structure.
